🔒 Security fix: Patch Reflected XSS in 404 template

🎯 **What:**
Fixed a potential Reflected XSS vulnerability in templates/404.php when outputting the scale abstract. The previous code was trimming words and stripping tags, but did not use WordPress's proper output escaping function.

⚠️ **Risk:**
If the tag stripping was bypassed or if the abstract contained malicious payloads that survived the initial sanitization, an attacker could potentially execute arbitrary scripts in the context of the user's browser when they visited a 404 page that displayed popular scale suggestions.

🛡️ **Solution:**
Wrapped the abstract output with esc_html(), which safely escapes characters such as <, >, &, ", and ', rendering any injected HTML or scripts as harmless text. The plugin version has also been incremented to 1.55.5 to reflect this patch.
